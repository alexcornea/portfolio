Little Dream Gardens

This is a website which I've built for my girlfriends hobby which is creating miniature gardens. She wanted to show her creations and also teach people how to do theyr own miniature gardens and other DIY projects. As this was my first website and because I was doing on my spare time it took me a few months to finish, but it was a really enjoyable experience.

After learning the Python basics, I've decided to check out Django. I really liked using it and because the documentation is quite easy to understand, I've decided to create the website with this great framework. One might ask why didn't I use WordPress, which is definitely a very good chose for this type of website. The reason for this is that I wanted to use a Python related framework as I wanted to further expand my knowledge of back-end with this language.

As for the database I used PostgreSQL as it seem a powerful, advanced and fairly easy to use option, especially if you use the pgAdmin 4 platform with it. The database stores the contents of the posts. For the front-end part I've used basic HTML and CSS with a little help from Bootstrap and for some elements I used JavaScript/jQuery. I did my best to make the website responsive, so it should work well on any device and browser.

After the website was ready and tested succesfully on local development server, I used Docker to create the image for it and also used NGINX for the web server part, as for the encryption, Let's Encrypt was used. For deployment I used AWS/EC2 and used Amazon RDS for the database side.

If you would like to visit my website to see what I've created, this is the address: www.littledreamgardens.com. To check the source code visit my GitHub page.
